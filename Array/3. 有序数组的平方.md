数组是有序的，平方之后仍然有序
## 暴力方法
## 双指针思路
```C++
vector<int> result;
k = num.size()-1;
for(i=0,j =num.size()-1;i<=j;)
{
    if(num[i]*num[i]>num[j]*num[j])
    {
        result[k--]= num[i]*num[i];
        i++;        
    }else{
        result[k--]= num[j]*num[j];
        j--;
    }
}
return result;
```
